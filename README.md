# Python Task Manager

A simple command-line task manager built in Python.
This project allows users to create, view, delete, and mark tasks as completed. Tasks are stored using JSON so they persist between sessions.

## Features

* Add new tasks
* View all tasks
* Delete tasks
* Mark tasks as completed or incomplete
* Tasks are saved to a `tasks.json` file
* Simple command-line interface

## Technologies Used

* Python 3
* JSON for data storage
* Git & GitHub for version control

## How to Run the Project

1. Clone the repository:

   git clone https://github.com/Leila-NM/python-projects.git

2. Navigate to the task manager folder:

   cd python-projects/task_manager

3. Run the program:

   python task_manager.py
